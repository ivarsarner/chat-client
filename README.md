# Web chat application

This is a web chat application which is divided in 2 parts:

- Client (this one): https://github.com/ivarsarner/chat-client
- Server: https://github.com/ivarsarner/chat-server

## Tech stack

- React
- Redux
- Websockets (Socket.io)
- Typescript

## Run locally

- npm start
